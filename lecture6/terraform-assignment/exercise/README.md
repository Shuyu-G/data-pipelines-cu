# Terraform Assignment: Your First Webserver on AWS

Deploy a simple HTML page to an AWS web server using Terraform, then retrieve the public URL.

## Objectives

1. Install Terraform on your machine
2. Deploy your first AWS web server with Terraform
3. Serve a custom HTML page with nginx
4. Retrieve the public link from Terraform output

## Prerequisites

- Terraform installed locally
- An AWS account
- AWS credentials configured locally, for example with environment variables or `aws configure`
- Permission to create EC2, VPC, subnet, route table, internet gateway, and security group resources

## Files

```text
terraform-assignment/
├── main.tf
├── README.md
└── index.html              # Generated by Terraform after apply
```

## What This Terraform Configuration Creates

- A dedicated VPC and public subnet
- An internet gateway and public route table
- A security group that allows inbound HTTP on port 80
- One Amazon Linux 2023 EC2 instance
- nginx installed by `user_data`
- A generated `index.html` file saved locally and copied onto the EC2 instance

## Step 1: Initialize Terraform

```bash
cd terraform-assignment
terraform init
```

If you want to use a different region, pass it as a variable:

```bash
terraform init
terraform apply -var="aws_region=us-east-1"
```

## Step 2: Deploy the Webserver

Run:

```bash
terraform apply
```

Type `yes` when prompted.

Terraform will:

- Generate `index.html`
- Build the AWS networking resources
- Launch an EC2 instance
- Install nginx
- Serve your HTML page over the public internet

## Step 3: Retrieve the Link

After `terraform apply`, Terraform prints outputs similar to:

```text
Outputs:

website_url = "http://3.120.45.67"
public_ip   = "3.120.45.67"
instance_id = "i-0123456789abcdef0"
```

Open `website_url` in your browser to verify the page is live.

You can print the URL again later with:

```bash
terraform output website_url
```

## Step 4: Clean Up

Destroy everything when you are done to avoid AWS charges:

```bash
terraform destroy
```

Type `yes` when prompted.
